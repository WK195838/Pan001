### 安裝必要的Python套件

```bash
# 安裝PDF處理套件（選擇其中一個）
pip install PyPDF2
pip install pdfplumber
pip install pymupdf
```

### 使用轉換腳本

```bash
# 轉換特定PDF檔案
python pdf_to_md.py GLATEST/11312損益表-月份.pdf

# 或直接執行（會自動處理預設檔案）

```
